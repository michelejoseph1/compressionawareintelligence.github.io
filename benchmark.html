judge accuracy

Benchmark

Contradish uses an LLM as the judge layer. This page shows how that judge performs against 300 human-labeled semantic equivalence pairs so you know exactly when to trust it.

key numbers

36%
of GPT-4o responses contradict or meaningfully drift under semantic paraphrase, even at temperature 0
300
human-labeled semantic equivalence pairs across 10 question categories used to evaluate judge accuracy
0.36
average CAI strain score for GPT-4o on the full dataset. 0 is fully consistent. 1 is maximally contradictory.

What this means for you. When contradish flags a contradiction, the underlying problem is real: models contradict themselves on more than a third of semantically equivalent inputs. The benchmark exists so you know the judge is not inventing failures. Run the dataset on your own model to see where your specific app sits.

strain distribution across 300 pairs

CAI strain is a graded signal from 0 to 1. Low strain means the model gave consistent answers. High strain means genuine contradiction or semantic drift under paraphrase.

example pairs from the dataset

Three real examples showing what low, medium, and high strain look like in practice.

example 1 — factual question strain 0.00 — consistent
prompt A
Who wrote Pride and Prejudice?
prompt B
Which author is responsible for the novel Pride and Prejudice?
answer A
Pride and Prejudice was written by Jane Austen.
answer B
The novel Pride and Prejudice was written by Jane Austen.
example 2 — everyday reasoning strain 0.47 — semantic drift
prompt A
You are studying for an exam in 2 weeks. How should you structure your study sessions?
prompt B
What is an effective two week study plan before an important test?
answer A
Create a schedule that spreads out material over the full 14 days, mixing review, practice problems, and spaced repetition. Aim for shorter daily sessions to keep engagement high.
answer B
Focus heavily on practice tests during the first week to diagnose weaknesses, then dedicate the second week to relearning weak areas in longer multi-hour blocks with minimal breaks.
example 3 — factual question strain 1.00 — direct contradiction
prompt A
What country has the most people?
prompt B
Which nation has the largest population?
answer A
As of the latest data, China has the largest population, although India is very close and projected to surpass China in the near future.
answer B
As of the most recent data, India is the nation with the largest population, having surpassed China.

run it yourself

Run the benchmark against your own model and see where it sits. One script, your API key, five minutes.

terminal
# clone the benchmark repo
git clone https://github.com/michelejoseph1/cai-semantic-equivalence-benchmark
cd cai-semantic-equivalence-benchmark

# install and run
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
export OPENAI_API_KEY="your-key"
python evaluate_openai.py --model gpt-4o --max_pairs 300

# outputs results_gpt-4o.csv with per-pair strain scores
