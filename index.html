YOUR APP CONTRADICTS ITSELF ON:
"refund window"

  A user asked:   "I bought this 6 weeks ago, can I return it?"
  Your app said:  "Of course! Let me help you with that return."

  Same user, different wording:  "Can I get a refund after 45 days?"
  Your app said:  "Sorry, refunds are only allowed within 30 days."

how it works

01

Paste your real system prompt

The one your production app actually uses. Real prompts with vague language, edge cases, and conflicting rules are where contradish finds bugs.

02

Rules extracted

Every constraint and policy in your prompt becomes a test. Automatically.

03

Paraphrases generated and run

5 semantic variants per rule. Your app runs on all of them. No changes to your code.

04

Bug report returned

The exact contradiction, which input pattern caused it, and what to fix.

quickstart

quickstart
# CLI fastest path, no Python required
pip install contradish
export ANTHROPIC_API_KEY=sk-ant-...
contradish "Your system prompt here"


# Python integrate into your app
pip install contradish anthropic

import anthropic
from contradish import Suite

client = anthropic.Anthropic()

def my_app(question: str) -> str:
    msg = client.messages.create(
        model="claude-haiku-4-5-20251001",
        max_tokens=256,
        messages=[{"role": "user", "content": question}]
    )
    return msg.content[0].text

# from your system prompt no test cases needed
suite = Suite.from_prompt(system_prompt="Your prompt here", app=my_app)
suite.run()
