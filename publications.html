Related work

External empirical validation

Emergent Misalignment: Narrow Finetuning Can Produce Broadly Misaligned LLMs

Betley et al. · 2025 · Peer-reviewed

Independent empirical confirmation of a core CAI prediction. The study shows that fine-tuning on insecure code with concealed intent produces misaligned behavior far outside the code domain. Misalignment disappears when the same data is framed with benign intent, showing that intention controls how compression strain propagates.

In CAI terms: a narrow contradiction in the training signal creates compression strain. That strain leaks across domains. The study matches the prediction that untracked compression produces global drift. It is exactly the pattern CAI calls compression strain memory.
